## Setup Instructions
1. Install dependencies:
   ```
   yarn install
   ```
   or
   ```
   npm install
   ```

2. Add required images to the public directory

3. Run development server:
   ```
   yarn dev
   ```
   or
   ```
   npm run dev
   ```

4. Build for production:
   ```
   yarn build
   ```
   or
   ```
   npm run build
   ```

5. Start production server:
   ```
   yarn start
   ```
   or
   ```
   npm start
   ```
